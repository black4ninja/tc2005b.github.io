Lab 26: Servicios web

Descripción

En esta actividad exploraremos algunos APIs de aplicaciones web, con el objetivo de que aprendas a integrar funcionalidad desarrollada por terceros en tus propias aplicaciones.

Objetivos de aprendizaje

Entender qué son los servicios web y su evolución.
Explorar APIs para aplicaciones web e integrar su funcionalidad en tus aplicaciones.

Instrucciones

  • Revisa junto con tu profesor, la presentación de Servicios Web.

    Revisa los siguientes enlaces a sitios de desarrolladores de APIs de algunas aplicaciones web:

    • Google maps
    • Google calendar
    • Facebook
    • Airtable
    • Riot Games
    • Twitter
    • Soundcloud
    • Vimeo
    • Instagram
    • Linkedin
    • Dropbox
    • Paypal

  • Selecciona uno de los APIs.

  • Consume el API dentro de una de tus aplicaciones web.

  • Responde a las preguntas dentro de tu aplicación.

Preguntas

  • ¿Qué ventajas y desventajas tiene la integración de tus aplicaciones web con servicios web desarrollados por terceros?

Laboratorio 1 Demo: Conociendo el ambiente de desarrollo Visual Studio .NET

Descripción de la actividad: En este primer laboratorio el profesor presentará a detalle el ambiente de desarrollo integrado de Visual Studio .NET. Además explicará la diferencia de paradigmas de programación de EOP (Programación Orientada a Eventos)  y OOP (Programación Orientada a Objetos).

Modalidad: Grupal

Familiarizarse con el ambiente de desarrollo Visual Studio .Net, así como con el lenguaje de programación C# .NET

Para ilustrar el uso de la herramienta, se creará desde el inicio una aplicación propuesta por el grupo.

Para tal fin, es necesario que leas los primeros capítulos de algún manual de referencia de dicha herramienta.

No es necesario entregar la aplicación construida, ya que es sólo un ejemplo académico.

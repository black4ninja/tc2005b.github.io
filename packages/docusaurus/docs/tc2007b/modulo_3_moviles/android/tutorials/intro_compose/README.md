# Introducción

El siguiente set de laboratorios son la base fundamental para hacer desarrollo Android utilizando como **interfaz Jetpack Compose** la forma más moderna utilizando un paradigma declarativo. También se utiliza como lenguaje de programación **Kotlin**.

Es importante que conozcas que actualmente existen varios caminos por los cuales se puede aprender Android, este es el más moderno, ya que muchos proyectos actuales siguen teniendo un gran impacto a partir de XML como modelo desarrollador de UI.

Para estos laboratorios puedes usar como guía la solución completa en este [repositorio](https://github.com/black4ninja/pokedex-android-2025).

Encontrarás que este [repositorio](https://github.com/black4ninja/pokedex-android-2025) contiene varios branches, los cuales son cada laboratorio por separado por lo que si por alguna razón tienes problemas al avanzar o quieres retomar desde un punto específico puedes hacerlo fácilmente.

Puedes también encontrar en el [repositorio](https://github.com/black4ninja/pokedex-android-2025) en la sección de **releases** los mismos compilados por laboratorio para descargar por zip si así lo prefieres.

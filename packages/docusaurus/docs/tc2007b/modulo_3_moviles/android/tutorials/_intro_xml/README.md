# Introducción

El siguiente set de laboratorios son la base fundamental para hacer desarrollo Android utilizando como **interfaz XML** o desarrollo por componentes de la forma más original en Android. También se utiliza como lenguaje de programación **Kotlin**.

Es importante que conozcas que actualmente existen varios caminos por los cuales se puede aprender Android, este es el más contemporáneo, ya que muchos proyectos actuales siguen teniendo un gran impacto a partir de XML como modelo desarrollador de UI.

La forma más actual es mediante el uso de Jetpack Compose el cual cambia ligeramente el paradigma de desarrollo de interfaces XML por un paradigma declarativo utilizando Kotlin directamente. Este paradigma lo veremos en otro tutorial más adelante, pero dado que muchos proyectos aún usan la estructura básica de XML aún hay mucho negocio, trabajo y esfuerzo por mantener y actualizar dichos proyectos.

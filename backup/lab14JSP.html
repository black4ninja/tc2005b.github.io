Actividad: Lab 14: JSP
Módulo: Paradigmas y aplicaciones cliente-servidor
Descripción
  • En esta actividad haremos una introducción a las Java Server Pages.
Modalidad
  • Individual.
Objetivos de la actividad
  • Que conozcas las tecnologías JEE
  • Realizar una aplicación con tecnologías JEE, utilizando JSP's.
Instrucciones
  • Revisa la presentación Introducción a JSP y realiza el ejercicio. Deberas entregarlo por Blackboard.
  • Realiza una aplicación sencilla utilizando JSP's. Puedes tomar como base alguno de tus laboratorios anteriores.
    • Recuerda que tu aplicación debe ser agradable para el usuario, y que las preguntas deben contestarse en alguna página dentro de tu sitio.
    • Si utilizaste alguna base de datos, aparte de los archivos de tu aplicación, entrega también el script de creación de la base de datos.

Preguntas

  • ¿Qué ventajas y desventajas tiene programar en una jsp sobre un servlet?
  • Describe un buen estilo de arquitectura de software utilizando jsp, servlets, y clases para interactuar con la base de datos. Relaciónalo con la arquitectura MVC. Puedes utilizar un diagrama. ¿Por qué es importante esta arquitectura? ¿Qué valor y qué desventajas presenta?
  • ¿Qué ventajas y desventajas tiene el enfoque JEE sobre el enfoque php con respecto a la implementación de una arquitectura de software MVC?

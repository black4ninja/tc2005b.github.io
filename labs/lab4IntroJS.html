Laboratorio 4: Fundamentos de JavaScript

Descripción

En esta actividad se hará una introducción a JavaScript.

Modalidad

Individual.

Objetivos de aprendizaje
  • Conocer el propósito y las características principales de JavaScript
  • Conocer las reglas básicas de sintaxis de JavaScript
  • Elaborar programas básicos con JavaScript
Instrucciones
  • Revisa la presentación Fundamentos de JavaScript

  • Escribe, prueba y corrige scripts de JavaScript para los siguientes problemas:

    • Funciones: Cuando se requiera escribir funciones, incluye un conjunto adecuado de casos de prueba.
    • Pruebas automatizadas: Utiliza console.assert() para automatizar las pruebas.
    • Resultados visibles: Permite al usuario ver los resultados en el documento HTML usando document.write(), modificando el DOM, o mostrando resultados en la consola.
    info ¿Qué es console.assert()?

    console.assert() es una función de depuración que verifica si una condición es verdadera. Si la condición es falsa, imprime un mensaje de error en la consola del navegador. Si es verdadera, no hace nada.

    Sintaxis:

    console.assert(condición, mensaje);

    Ejemplo de uso para pruebas:

    // Probar una función suma
function suma(a, b) {
    return a + b;
}

console.assert(suma(2, 3) === 5, "Error: suma(2,3) debería ser 5");
console.assert(suma(-1, 1) === 0, "Error: suma(-1,1) debería ser 0");
console.assert(suma(0, 0) === 0, "Error: suma(0,0) debería ser 0");

// Si todas las condiciones son verdaderas, no verás nada en la consola
// Si alguna es falsa, verás el mensaje de error correspondiente

    Consejo: Abre la consola del navegador (F12 o Ctrl+Shift+I) para ver los resultados de las pruebas.

    1. Ejercicio 1: Tabla de cuadrados y cubos
      Entrada: Un número pedido con prompt (por ejemplo: 5).
      Salida: Una tabla con los números del 1 al número dado con sus cuadrados y cubos. Utiliza document.write para producir la salida.
      Ejemplo: Si el usuario ingresa 3, la salida debe mostrar una tabla con columnas: Número | Cuadrado | Cubo, con los valores 1-3.
    2. Ejercicio 2: Quiz de suma con tiempo
      Entrada: Usando un prompt se pide el resultado de la suma de 2 números generados de manera aleatoria.
      Salida: La página debe indicar si el resultado fue correcto o incorrecto, y el tiempo que tardó el usuario en responder (en segundos).
      Sugerencia: Usa Date.now() para medir el tiempo al inicio y al final.
    3. Ejercicio 3: Contador de números
      Función: contador
      Parámetros: Un arreglo de números.
      Regresa: Un objeto con tres propiedades: negativos, ceros, y positivos.
      Ejemplo: contador([1, -2, 0, 5, -3, 0, 7]) debe regresar {negativos: 2, ceros: 2, positivos: 3}
    4. Ejercicio 4: Promedios de matriz
      Función: promedios
      Parámetros: Un arreglo de arreglos de números (matriz).
      Regresa: Un arreglo con los promedios de cada uno de los renglones de la matriz.
      Ejemplo: promedios([[10, 20, 30], [5, 15, 25]]) debe regresar [20, 15]
    5. Ejercicio 5: Número inverso
      Función: inverso
      Parámetros: Un número.
      Regresa: El número con sus dígitos en orden inverso.
      Ejemplo: inverso(12345) debe regresar 54321, inverso(1000) debe regresar 1
    6. Ejercicio 6: Problema personalizado con Programación Orientada a Objetos
      Crea una solución para un problema de tu elección. El problema debe estar descrito en un documento HTML, y la solución implementada en JavaScript usando Programación Orientada a Objetos.
      Requisitos:
      • Crear al menos una clase/objeto con constructor
      • El objeto debe tener al menos 2 métodos además del constructor
      • Mostrar los resultados en el documento HTML

      Ideas de problemas: Sistema de gestión de tareas, calculadora de propinas, conversor de unidades, juego simple (piedra-papel-tijera), registro de gastos personales, etc.
Preguntas a responder
  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
  • ¿Cómo se declara una variable con alcance local dentro de una función?
  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

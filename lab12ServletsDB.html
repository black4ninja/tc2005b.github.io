Actividad: Lab 15: JEE y Bases de datos
Módulo: Paradigmas y aplicaciones cliente-servidor
Descripción
  • En esta actividad interactuaremos con una base de datos utilizando tecnologías JEE.
Modalidad
  • Individual.
Objetivos de la actividad
  • Que conozcas las tecnologías JEE
  • Realizar una aplicación con tecnologías JEE
  • Interactuar con una base de datos a través de tecnologías JEE
Instrucciones
  • Realiza los siguientes tutoriales:
    • https://netbeans.org/kb/docs/web/quickstart-webapps.html
    • https://netbeans.org/kb/docs/web/mysql-webapp.html
  • Revisa junto con tu profesor las siguientes presentaciones:
    • Introducción a jsp
    • Java beans, JSTL, EL
    • Servlets y bases de datos
  • Agrega funcionalidad a la aplicación del tutorial donde se interactúa con mySql (o si lo prefieres inicia una nueva aplicación o utiliza otra aplicación). Entre la funcionalidad que agregues debes insertar o modificar algún registro de la base de datos.
    • Recuerda que tu aplicación debe ser agradable para el usuario, y que las preguntas deben contestarse en alguna página dentro de tu sitio.
    • Aparte de los archivos de tu aplicación, entrega también el script de creación de la bd de datos.

Preguntas

  • ¿Qué diferencias en cuanto a desempeño observas con respecto a php? ¿a qué se deben estas diferencias?
  • ¿Qué ventajas observas que tiene JEE sobre php?
  • Pensando en tu proyecto anterior, y considerando que ahora conoces la tecnología JEE, ¿en qué lo desarrollarías y por qué razones?
